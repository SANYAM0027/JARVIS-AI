## 📌 Project Overview

JARVIS AI is an advanced voice-controlled virtual assistant and futuristic analytics dashboard inspired by Stark Industries technology.

The system combines conversational AI, persistent memory, speech recognition, text-to-speech synthesis, macOS automation, and a real-time web analytics HUD into a single intelligent desktop application.

The project demonstrates practical applications of Artificial Intelligence, desktop automation, multithreading, voice interfaces, and data visualization using Python.

---

## 🎯 Problem Statement

Traditional desktop assistants often lack personalization, persistent memory, and advanced automation capabilities.

This project addresses the following challenges:

* How can voice interaction be made more natural and responsive?
* How can an AI assistant remember information across sessions?
* How can website analytics be visualized in a futuristic dashboard?
* How can repetitive desktop tasks be automated through voice commands?
* How can GUI responsiveness be maintained during continuous speech processing?

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/SANYAM0027/JARVIS-AI.git
cd JARVIS-AI
```

### Install Dependencies

```bash
pip install PyQt6 pyqtgraph sounddevice numpy requests python-dotenv openai
```

### Configure Environment Variables

Create a `.env` file:

```env
OPENROUTER_API_KEY=your_api_key
JARVIS_MODEL=qwen/qwen-2.5-7b-instruct:free
```

---

## ▶️ Execution

Run the application:

```bash
python3 main.py
```
